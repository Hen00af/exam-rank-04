# Exam-Rank-04

This repository contains solutions and subjects for the 42 Common Core Level 5 Exam (Exam-Rank-04).

## Directory Structure

- `Level 1/`
  - `ft_popen/`: Subject and solution for the ft_popen project
  - `picoshell/`: Subject and solution for the picoshell project
  - `sandbox/`: Subject and solution for the sandbox project
- `Level 2/`
  - `argo/`: Subject and solution for the argo project
  - `vbc/`: Subject and solution for the vbc project

Each subdirectory contains a `subject.txt` file describing the project requirements. Some may also include header files or source code.
